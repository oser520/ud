A Chess Engine

Summary

Designed a chess engine from scratch for fun! The engine can play thousands of simultaneous games against bots or homo sapiens, and uses modern AI search algorithms to analyse millions of moves per second efficiently, in terms of both speed and space. The engine is designed to be flexiable, and thus it is easy to plug in components that use different strategies, such as for evaluating board positions, or searching the space of moves. The engine is compliant with well known chess standards, such as the Chess Engine Communication Protocol and the Universal Chess Interface, effectively making it plug-and-play with chess GUIs.

Technology

C++ Artificial Intelligence Search Algorithms Evaluation Functions Multi-threading

A Movie Trailer Website

Summary

Created a python script to generate a website of movie trailers and movie data. The movies that should be displayed in the website can be specified by the user of the program, and all of the data for the website is fetched from the Web upon initial request, and thereafter fetched from a database if it is fetching data for the same movie for a second time.

Data Sources

  • Youtube
  • OMDb (Open Movie Database)
  • IMDb (Internet Movie Database)

Technology

Python HTML Javascript Bootstrap JQuery Regular Expressions Web APIs
